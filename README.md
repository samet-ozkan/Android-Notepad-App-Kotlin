# Android-Notepad-App-Kotlin
Kotlin (MVVM) Android not defteri uygulaması.

---
## Özellikler
* Not ekleme, düzenleme ve silme
* Notları etiketleme ve renklendirme
* Notları arama ve etiketlere göre filtreleme

---

## Kullanılan Teknolojiler
* Room Database
* Coroutine
* Dagger-Hilt
* ViewModel
* LiveData
* ViewBinding
